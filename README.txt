How to run the project


Option 1: Run the source code locally (development mode):

1- you need nodejs installed on your machine

2- open GenAI folder on 2 terminals:

* Terminal 1:

cd backend

npm run dev

* Terminal 2:

cd frontend

npm run dev

3- browse the link: http://localhost:5173/



Option 2: Simply visit the deployed production version on: https://gen-ai-frontend-omega.vercel.app/
